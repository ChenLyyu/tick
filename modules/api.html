API Reference¶

This is the full class and function references of tick. Please look at the modules documentation cited below for more examples and use cases, since direct class and function API is not enough for understanding their uses.

tick.inference: Inference classes¶

This module contains all classes giving inference tools, intended for end-users.

User guide: See the tick.inference: user-friendly inference tools section for further details.

Generalized linear models¶

inference.LinearRegression([fit_intercept, …]) Linear regression learner, with many choices of penalization and solvers.
inference.LogisticRegression([…]) Logistic regression learner, with many choices of penalization and solvers.
inference.PoissonRegression([step, …]) Poisson regression learner, with exponential link function.

Survival Analysis¶

inference.nelson_aalen(timestamps, …) Computes the Nelson-Aalen cumulative hazard rate estimation
inference.kaplan_meier(timestamps, …) Computes the Kaplan-Meier survival function estimation
inference.CoxRegression([penalty, C, …]) Cox regression learner, using the partial Cox likelihood for proportional risks, with many choices of penalization.

Hawkes¶

inference.HawkesExpKern(decays[, gofit, …]) Hawkes process learner for exponential kernels with fixed and given decays, with many choices of penalization and solvers.
inference.HawkesSumExpKern(decays[, …]) Hawkes process learner for sum-exponential kernels with fixed and given decays, with many choices of penalization and solvers.
inference.HawkesEM([kernel_support, …]) This class is used for performing non parametric estimation of multi-dimensional Hawkes processes based on expectation maximization algorithm.
inference.HawkesADM4(decay[, C, …]) A class that implements parametric inference for Hawkes processes
inference.HawkesBasisKernels(kernel_support) This class is used for performing non parametric estimation of multi-dimensional Hawkes processes based on expectation maximization algorithm and the hypothesis that kernels are linear combinations of some basis kernels.
inference.HawkesSumGaussians(max_mean_gaussian) A class that implements parametric inference for Hawkes processes
inference.HawkesConditionalLaw([delta_lag, …]) This class is used for performing non parametric estimation of multi-dimensional marked Hawkes processes based on conditional laws.

optim.model.ModelPoisReg(…) Poisson regression model with identity or exponential link.
optim.model.ModelCoxRegPartialLik() Partial likelihood of the Cox regression model (proportional hazards).

optim.model.ModelSCCS(n_intervals: int, …) SCCS model.

tick.optim.prox: Proximal operators classes¶

This module contains all the proximal operators available in tick.

User guide: See the 2. tick.optim.prox: proximal operators section for further details.

optim.prox.ProxZero(range: tuple = None) Proximal operator of the null function (identity)
optim.prox.ProxL1(strength: float, …) Proximal operator of the L1 norm (soft-thresholding)
optim.prox.ProxL1w(strength: float, …) Proximal operator of the weighted L1 norm (weighted
optim.prox.ProxElasticNet(strength: float, …) Proximal operator of the ElasticNet regularization.
optim.prox.ProxL2Sq(strength: float, …) Proximal operator of the squared L2 norm (ridge penalization)
optim.prox.ProxL2(strength: float, …) Proximal operator of the L2 penalization.
optim.prox.ProxMulti(proxs: tuple) Multiple proximal operator.
optim.prox.ProxNuclear(strength: float, …) Proximal operator of the nuclear norm, aka trace norm
optim.prox.ProxPositive(range: tuple = None, …) Projection operator onto the half-space of vectors with
optim.prox.ProxEquality(strength: float = 0, …) Projection operator onto the set of vector with all coordinates equal (or in the given range if given one).
optim.prox.ProxSlope(strength: float, …) Proximal operator of Slope penalization.
optim.prox.ProxTV(strength: float, …) Proximal operator of the total-variation penalization
optim.prox.ProxBinarsity(strength: float, …) Proximal operator of binarsity.
optim.prox.ProxGroupL1(strength: float, …) Proximal operator of group-L1, a.k.a group-Lasso.

tick.optim.solver: Solver classes¶

This module contains all the solvers available in tick.

User guide: See the 3. tick.optim.solver: solvers section for further details.

Batch solvers¶

optim.solver.GD(step: float = None, …) GD (proximal gradient descent) algorithm.
optim.solver.AGD(step: float = None, …) AGD (accelerated proximal gradient descent) algorithm.
optim.solver.BFGS(tol: float = 0.0, …) BFGS (Broyden, Fletcher, Goldfarb, and Shanno ) algorithm.
optim.solver.GFB(step: float = None, …[, …]) Generalized Forward-Backward algorithm

Stochastic solvers¶

optim.solver.SGD(step: float = None, …) Stochastic Gradient Descent solver
optim.solver.SVRG(step: float = None, …) Stochastic variance reduced gradient
optim.solver.SDCA(l_l2sq: float, …) Stochastic Dual Coordinate Ascent solver
optim.solver.AdaGrad(step: float = 0.01, …) Adaptive Gradient Descent solver

History¶

optim.history.History() A class to manage the history along iterations of a solver

tick.preprocessing: Preprocessing utilities¶

This module contains some utilities functions for preprocessing of data.

User guide: See the tick.preprocessing: preprocessing utilities section for further details.

Classes¶

preprocessing.FeaturesBinarizer([method, …]) Transforms continuous data into bucketed binary data.
