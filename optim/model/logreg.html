Logistic regression model¶

class mlpp.optim.model.ModelLogReg(fit_intercept: bool = True, n_threads: int = 1)[source]¶

Logistic regression model. This class gives first order information (gradient and loss) for this model

Parameters:

fit_intercept : bool

If True, the model uses an intercept

Attributes

labels (numpy.ndarray, shape=(n_samples,) (read-only)) The labels vector
n_samples (int (read-only)) Number of samples
n_features (int (read-only)) Number of features
n_coeffs (int (read-only)) Total number of coefficients of the model
n_threads (int, default=1 (read-only)) Number of threads used for parallel computation. * if int <= 0: the number of physical cores available on the CPU * otherwise the desired number of threads
