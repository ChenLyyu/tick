Solvers [mlpp.optim.solver]¶

Introduction¶

Module that contains all solvers available in MLPP. This module is used throughout the library. “Batch” or “full” solvers use a full pass over data at each iteration, while “stochastic” solvers make epoch_size iterations using a single data-point at each iteration.

Batch solvers¶

Solver Class
Proximal gradient descent Ista
Accelerated proximal gradient descent Fista
BFGS (Broyden, Fletcher, Goldfarb, and Shanno) BFGS
Self Concordant Proximal Gradient SCPG

Stochastic solvers¶

Solver Class
Stochastic gradient descent SGD
Stochastic variance reduced gradient descent SVRG
Stochastic dual coordinate ascent SDCA
