class mlpp.base.TimeFunction(values, border_type: int = 0, inter_mode: int = 0, dt: float = 0, border_value: float = 0)[source]¶

A function depending on time.

It is causal as its value is zero for all \(t < 0\).

Parameters:
  • values – Either a number if the TimeFunction is constant or a list of two numpy arrays [T, Y] where Y are the values taken by the TimeFunction at times T
  • border_type –

    Handle the values returned after the after the last given t (non constant case)

    • Border0 : value will be \(0\)
    • BorderConstant : value will be given by border_value
    • BorderConstant : value will equal to the last known value
  • inter_mode –

    Handle the way we extrapolate between two known values (non constant case)

    • InterLinear : value will be linearly interpolated following the formula \(f(x) = \frac{y_{t+1} - y_{t}}{x_{t+1} - x_{t}}\)
    • InterConstLeft : value will be equal to the next known point
    • InterConstRight : value will equal to the previous known point
  • dt – The value used for the sub-sampling. If left to 0, it will be assigned automatically to a fifth of the smallest distance between two points
  • border_value – See border_type, BorderConstant case

Note

TimeFunction are made to be every efficient when call if to get a specific value (\(\mathcal{O}(1)\)), however this leads us to have it taking a lot of space in memory.
