Proximal operators [mlpp.optim.prox]¶

Introduction¶

Module for the computation of proximal operators. This module is heavily used throughout the library, and mostly called from within solvers of the module mlpp.optim.solver.

The proximal operator of a convex function \(g : \mathbb R^d \rightarrow \mathbb R^d\) at some point \(y\in \mathbb R^d\) is defined as the unique minimizer of the problem

\[\text{prox}_{g}(\theta, t) =\text{argmin}_{\theta' \in \mathbb R^d} \Big\{ \frac 12 \| \theta - \theta' \|_2^2 + t g(\theta') \Big\}\]

where \(\lambda > 0\) is a regularization parameter. Note that in the particular case where \(g(\theta) = \delta_{C}(\theta)\), with \(C\) a convex set, then \(\text{prox}_g\) is a projection operator (here \(\delta_{C}(\theta) = 0\) if \(\theta \in C\) and \(+\infty\) otherwise).

Available operators¶

Operator Formula Class
Null penalization \(g(\theta) = 0\) ProxZero
Projection onto non-negative vectors \(g(\theta) = \delta_C(\theta)\) where \(C=\) set of vectors with non-negative entries ProxPositive
penalization \(g(\theta) = \sum_{j=1}^d |\theta_j|\) ProxL1
weighted penalization \(g(\theta) = \sum_{j=1}^d w_j |\theta_j|\) ProxL1w
Ridge penalization \(g(\theta) = \sum_{j=1}^d \frac{\theta_j^2}{2}\) ProxL2Sq
ElasticNet penalization \(g(\theta) = \sum_{j=1}^{d} \alpha |\theta_j| + (1 - \alpha) \frac{\theta_j^2}{2}\) ProxElasticNet
Total-variation penalization \(g(\theta) = \sum_{j=2}^d |\theta_j - \theta_{j-1}|\) ProxTV
Nuclear-norm penalization \(g(\theta) = \sum_{j=1}^{q} \sigma_j(\theta)\) ProxNuclear
Sorted-L1 penalization \(g(\theta) = \sum_{j=1}^{d} w_j |\theta_{(j)}|\) where \(|\theta_{(j)}|\) is decreasing ProxSortedL1

Example¶

The next example shows a basic use of a proximal operator.:

import numpy as np
from bokeh.plotting import output_notebook, show
from mlpp.optim.prox import ProxTV
from mlpp.plot import stems

output_notebook()

n_coeffs = 100
x = np.random.randn(n_coeffs)
prox = ProxTV(0.5)
stems([x, prox.call(x)])
