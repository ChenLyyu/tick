Hawkes likelihood model with exponential kernels and with decrease¶

class mlpp.optim.model.ModelHawkesFixedExpKernLogLik(decays: numpy.ndarray, n_threads: int = -1)[source]¶

Model for Hawkes processes with fixed exponential kernels using the log-likelihood contrast. For now, all kernels are equal to an exponential and a fixed decay across nodes.

Parameters:

decays : float

The decay coefficient of the exponential kernel

n_threads : int, default=-1 (read-only)

Number of threads used for parallel computation.

  • if int <= 0: the number of physical cores available on the CPU
  • otherwise the desired number of threads

Attributes

n_nodes (int (read-only)) Number of components, or dimension of the Hawkes model
data (list of numpy.array (read-only)) The events of each component of the Hawkes. Namely data[j] contains a one-dimensional numpy.array of the events’ timestamps of component j
