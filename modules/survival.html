tick.survival¶

This module provides some tools for survival analysis. It provides a learner for Cox regression and a model for Self Control Case Series (SCCS).

1. Learners¶

For now, this module provides only a learner for Cox regression for proportional hazards, using the partial likelihood. It also provides Nelson-Aalen and Kaplan-Meier estimators.

CoxRegression([penalty, C, solver, step, …]) Cox regression learner, using the partial Cox likelihood for proportional risks, with many choices of penalization.
nelson_aalen(timestamps, event_observed) Computes the Nelson-Aalen cumulative hazard rate estimation
kaplan_meier(timestamps, event_observed) Computes the Kaplan-Meier survival function estimation

2. Models¶

The following models can be used with solvers for inference. Note that ModelCoxRegPartialLik cannot be used for now with a stochastic solver, while we can with ModelSCCS.

ModelCoxRegPartialLik() Partial likelihood of the Cox regression model (proportional hazards).
ModelSCCS(n_intervals: int, n_lags: int) SCCS model.

3. Simulation¶

We provide tools for the simulation of datasets, for both right-censored durations following a proportional risks model, and a SCCS model. SimuCoxReg generates data in the form of i.i.d triplets \((x_i, t_i, c_i)\) for \(i=1, \ldots, n\), where \(x_i \in \mathbb R^d\) is a features vector, \(t_i \in \mathbb R_+\) is the survival time and \(c_i \in \{ 0, 1 \}\) is the indicator of right censoring. Note that \(c_i = 1\) means that \(t_i\) is a failure time while \(c_i = 0\) means that \(t_i\) is a censoring time.

SimuCoxReg(coeffs: numpy.ndarray, …) Simulation of a Cox regression for proportional hazards
SimuSCCS(n_samples, n_intervals, n_features, …) Simulation of a Self Control Case Series (SCCS) model.

Example¶

"""
==============================
Cox regression data simulation
==============================

Generates Cox Regression realization given a weight vector
"""

import matplotlib.pyplot as plt
import numpy as np
from tick.survival import SimuCoxReg

n_samples = 150
weights = np.array([0.3, 1.2])

seed = 123
simu_coxreg = SimuCoxReg(weights, n_samples=n_samples, seed=123, verbose=False)
X, T, C = simu_coxreg.simulate()

plt.figure(figsize=(6, 4))

plt.scatter(*X[C == 0].T, c=T[C == 0], cmap='RdBu', marker="x",
            label="censoring")
plt.scatter(*X[C == 1].T, c=T[C == 1], cmap='RdBu', marker="o",
            label="failure")
plt.colorbar()
plt.legend(loc='upper left')
plt.title('Cox regression', fontsize=16)
